# RisenIOT 物联网云平台

基于 GoFrame 框架构建的一款物联网云平台应用

## 接入方式支持

- [X] HTTP 协议接入
- [ ] MQTT 协议接入
- [ ] UDP 协议接入

## 被动设备发现协议支持

- [ ] SDAP 海康设备发现协议

## 设备协议支持

- [X] 平台 HTTP 开放接入协议
